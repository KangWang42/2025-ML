# 1 课程内容

该库为中山大学星星点灯课题组-机器学习学习班资料库

## 1.1 课程要求 
- **课程目标**：掌握机器学习与深度学习核心模型的基本原理、代码实现、结果报告，提升复杂医学数据建模能力与科研实践水平。 
- **课程形式**：本课程共16讲（48学时），采用问题驱动教学法（医学问题→传统方法局限→代码实现→方法原理）。每节课须提前准备PPT、典型案例、完整代码（有详细注释）

## 1.2 课程大纲
| 周次 | 日期 | 星期 | 讲授内容 | 课时 |
| :---: | :---: | :---: | :--- | :---: |
| 1 | | | **第一讲：机器学习核心框架与数学基础**<br> 概述：机器学习任务分类、技术实现流程；<br> 机器学习、深度学习对比：传统机器学习与深度学习差异及其在医学场景的应用差异；<br> 数学基础模块：多元函数梯度、矩阵运算、最优化理论基础。 | 3 |
| 2 | | | **第二讲：EDA（探索性数据分析）与数据预处理**<br> EDA的目的；<br> EDA的内容：描述性统计、单变量分析、多变量分析、异常值分析、缺失值分析；<br> 数据预处理的目的；<br> 数据预处理的内容：缺失值处理、异常值处理、重复值处理、定量数据转换（标准化/归一化/对数转换等）、分类数据编码、数据合并（内连接、左连接等方式）。 | 3 |
| 3 | | | **第三讲：评价指标体系**<br> 分类任务指标；<br> 回归任务指标；<br> 交叉验证；<br> 偏差-方差权衡与过拟合。 | 3 |
| 5 | | | **第四讲：决策树与随机森林**<br>**问题**：如何构建一个可解释性强、能自动处理变量交互和非线性关系、且对缺失值/异常值有一定稳健性的预测模型？<br>**传统局限**：线性模型难以捕捉复杂交互；手动构建规则模型耗时且可能不全面；某些统计模型对数据分布假设严格。<br>**理论**：<br> 决策树模型原理：ID3/C4.5/CART、算法对比、预剪枝与后剪枝数学原理；<br> 随机森林算法：Bagging集成框架、OOB误差估计、特征重要性度量方法。 | 3 |
| 6 | | | **第五讲：传统分类模型**<br>**问题1**：在急诊分诊中，患者症状描述（如：发热≥39℃、干咳、淋巴细胞减少）可能涉及数百个特征，但实际组合稀疏（多数症状未出现）。如何快速计算其患流感、新冠或细菌性肺炎的概率？<br>**传统局限**：逻辑回归在高维稀疏特征导致模型估计不稳定；决策树无法输出校准概率且对特征缺失敏感。<br>**问题2**：对于某些罕见病，现有模型因样本不足无法训练，如何基于历史病历库动态匹配最接近的患者案例辅助临床决策？<br>**传统局限**：传统模型需要足够的阳性样本用于训练。<br>**理论**：<br> 朴素贝叶斯原理及变体、拉普拉斯平滑算法；<br> K近邻算法核心技术。 | 3 |
| 7 | | | **第六讲：支持向量机与核方法**<br>**问题**：如何在特征空间复杂（线性不可分）、样本量相对较小的情况下，构建一个泛化能力强的稳健分类器？<br>**传统局限**：线性分类器（如逻辑回归）无法处理复杂边界；其他非线性模型可能在小样本下容易过拟合。<br>**理论**：<br> 线性SVM理论：最优超平面推导、合页损失函数与支持向量几何意义；<br> 核方法核心技术：Mercer定理、核函数选择及计算复杂度优化。 | 3 |
| 8 | | | **第七讲：无监督学习**<br>**问题1（聚类）**：如何在无已知标签的情况下，从患者群体数据中发现潜在亚型？<br>**问题2（降维）**：如何可视化高维医学数据（如基因表达数据）以探索结构、识别离群点？<br>**理论**：<br> 聚类分析：K-means、层次聚类算法原理及优化策略；<br> 降维：PCA、t-SNE、UMAP原理与应用场景对比。 | 3 |
| 9 | | | **第八讲：梯度提升树与集成学习**<br>**问题**：如何系统整合多个弱模型提高模型预测精度？<br>**传统局限**：单一模型的预测精度可能有限。<br>**理论**：<br> 梯度提升核心理论：加法模型框架、损失函数优化及正则化策略；<br> 模型对比分析：XGBoost、LightGBM、CatBoost；<br> 集成学习基础：Boosting与Bagging对比、Stacking融合策略。 | 3 |
| 10 | | | **第九讲：机器学习总结与深度学习基础**<br> 传统机器学习模型总结；<br> Python基础；<br> 深度学习基础理论：感知机模型、神经网络架构、损失函数与隐藏层概念；<br> CPU和GPU介绍。 | 3 |
| 11 | | | **第十讲：人工神经网络基础**<br> 神经网络的学习；<br> 误差反向传播算法：计算图构建、链式法则；<br> 优化器、权重初始值、Batch Normalization。 | 3 |
| 12 | | | **第十一讲：卷积神经网络**<br>**问题**：如何让计算机自动从医学影像中识别病灶、进行分类或分割？<br>**理论**：<br> CNN基本结构、卷积层和池化层意义、经典模型设计思想对比（VGG、ResNet、DenseNet等）；<br> 迁移学习简介。 | 3 |
| 13 | | | **第十二讲：循环神经网络**<br>**问题**：如何建模医学时序数据，如患者连续的生理指标监测中的动态依赖关系？<br>**传统局限**：ARIMA需进行平稳性检验、参数识别等人工预处理步骤，且仅能捕捉短期线性依赖，对复杂模式建模能力弱。<br>**理论**：<br> RNN基本结构和原理；<br> LSTM和GRU门控机制原理。 | 3 |
| 14 | | | **第十三讲：注意力机制**<br>**问题**：如何让模型在处理序列数据或图像时，能够“聚焦”于最重要的部分？<br>**理论**：<br> 基础理论：注意力评分函数、Bahdanau注意力、自注意力机制、多头注意力模型架构；<br> 技术扩展：位置编码方法、Transformer模型。 | 3 |
| 15 | | | **第十四讲：特征选择与模型优化**<br>**问题**：面对高维生物医学数据，如何筛选出最相关、最有预测力的特征，提升模型可解释性与泛化能力？如何系统地调整模型参数和结构，以获得最佳性能并防止过拟合？<br>**理论**：<br> 特征选择方法：过滤法、包装法、嵌入法；<br> 模型优化技术：超参数调优策略、正则化方法及Dropout技术实现。 | 3 |
| 16 | | | **第十五讲：可解释机器学习核心方法**<br>**问题**：在医疗决策中，模型预测结果的可信度至关重要。如何理解复杂模型为何做出某个预测？如何让“黑箱”模型透明化？<br>**理论**：<br> 全局解释和局部解释的概念；<br> 常用可解释机器学习方法：SHAP、LIME、特征重要性、部分依赖图等。 | 3 |
| 17 | | | **第十六讲：深度学习总结与项目实战**<br> 深度学习总结；<br> 完整建模流程：通过两个项目分别总结传统机器学习和深度学习建模过程。 | 3 |
